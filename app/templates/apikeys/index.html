📖 Quick Start

Use your API key with any OpenAI-compatible client. Set the base URL to:

https://ecolyxis.co.uk/v1

Python (openai SDK)

from openai import OpenAI

client = OpenAI(
    base_url="https://ecolyxis.co.uk/v1",
    api_key="ecolyx_your_key_here"
)

response = client.chat.completions.create(
    model="ecolyxis",
    messages=[{"role": "user", "content": "Hello!"}]
)
print(response.choices[0].message.content)

curl

curl https://ecolyxis.co.uk/v1/chat/completions \
  -H "Authorization: Bearer ecolyx_your_key_here" \
  -H "Content-Type: application/json" \
  -d '{"model":"ecolyxis","messages":[{"role":"user","content":"Hello!"}]}'

Streaming

response = client.chat.completions.create(
    model="ecolyxis",
    messages=[{"role": "user", "content": "Hello!"}],
    stream=True
)
for chunk in response:
    print(chunk.choices[0].delta.content or "", end="")

Limits

  • 30 requests per minute
  • 60 messages per minute
  • 500,000 tokens per day (resets midnight UTC)
  • Max 5 API keys per account
